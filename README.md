# Overview

Being a software engineer, I am enhancing my expertise with the help of Kotlin, a programming language that operates on the Java Virtual Machine (JVM). The given introductory project is my initial experience with developing a Kotlin program, which will provide me with a working development environment and ensure that I am able to write, compile, and execute Kotlin code on my workstation.

It is a simple Hello, World! application program written in Kotlin. It illustrates the simplest arrangement of a Kotlin application with the entry point of a Kotlin application as the main function and the standard output in the form of println.

The goal of my creation of this software is to install and test my Kotlin development environment, learn the basics of the Kotlin compilation process, and establish a base on which future and more complex Kotlin applications will be built.

[Software Demo Video](http://youtube.link.goes.here)

# Development Environment

I developed this software on macOS using the VSCode IDE. The Kotlin compiler (`kotlinc`) and the OpenJDK runtime were installed via Homebrew. The program is compiled into a JAR file using `kotlinc` with the `-include-runtime` flag, and executed with the `java -jar` command.

The programming language used is Kotlin 2.3.10 and running on OpenJDK 25.0.2. Kotlin is a statically typed, cross-platform language developed by JetBrains that is fully interoperable with Java and compiles to JVM bytecode.
